# Task Manager — Project & Team Management

## Overview

Task Manager is a full-stack web application for managing projects, tasks, and teams. It provides role-based access control, 
secure authentication via Keycloak, and file storage for identity documents.

Live application: https://task-management-rose-iota.vercel.app

---

## Features

- User management with role-based access control (Admin, Responsable, User)
- Project and task creation, assignment, and status tracking
- Identity document upload stored securely in RustFS
- Real-time dashboard.
- Full OAuth2/JWT authentication via Keycloak

---

## Technologies

**Backend** — Java 25, Spring Boot 4, Keycloak , PostgreSQL, RustFS.

**Frontend** — Angular 21.

**Infrastructure** — Docker Compose, Nginx, Oracle Cloud, Vercel
